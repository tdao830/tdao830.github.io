Data Visualization

This project was taking data from the Chicago's data portal and projecting that data in different forms.For this specific project, the occurances of service requests for potholes was analyzed and displayed in a table, pie chart, and google maps heat map form.These were all created using google's charts and ajax calls were made with queries to grab the data from the Chicago database.

Charts

The Table simply displays the number of potholes per community area which was given a specific number.
The information is also displayed in a Pie Chart form, with different colors for the different community areas.
The Heat Map displays these potholes using google maps api.
